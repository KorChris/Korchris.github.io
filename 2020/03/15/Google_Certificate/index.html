Google Certificate for ML

구글 Certificate에서 요구하는 능력은 다음과 같습니다.

(합격기준 100점만점 90점 이상)

  1. Basic / Simple model
  2. Model from learning dataset
  3. CNN with real world image dataset
  4. NLP text classification with real world text dataset
  5. Sequence model with real world numeric dataset

시험에 참여하기 위해서는 다음을 만족해야합니다.

  1. ML과 DL의 중요한 개념들을 사용할 줄 알아야함
  2. Tensorflow를 사용해서 ML models를 만들어야함
  3. DNN, CNN을 이용하여 Image Recognition을 할줄 알며, 현실이미지를 다루는 것에 익숙하며 loss, accuracy를 plot하고 overfitting을 방지하는 전략들을 알아야함(augmentation, dropout등)
  4. Image Recognition뿐만 아니라, object detection, text recognition을 사용할 줄 알아야함(RNN(GRU, LSTMS)을 위한 text tokenization 포함)
  5. NLP 문제를 해결하기 위한 neural network를 설계할 줄 알아야함.(using Tensorflow)

시험 환경은 다음과 같습니다.

  1. 파이참 IDE를 사용하여 인터넷이 되는 어디든 본인의 컴퓨터로 시험을 볼 수 있습니다.
  2. 시험은 온라인으로 진행되며, 파이참을 통해 tensorflow를 이용합니다.
  3. 시험은 5시간동안 진행됩니다.
  4. 시험을 치르기 위해 파이참 IDE를 이용하여 Tensorflow Exam Plugin을 설치하게 됩니다. 파이참 IDE를 이용하는 것에 익숙해지기를 추천합니다.
  5. 5시간이 지나기 전에 시험을 완료했다면, 일찍 submit하실 수 있습니다. 만약 5시간이 지나게 되면, 자동적으로 submit이 진행됩니다.
  6. 이 문서를 모두 읽어보셨다면, Exam URL에 방문할 수 있습니다.

참여자 identification and authentication

여권/운전면허증/주민등록증 중 1개 (primary)

시험 관련

  1. 시험을 보는 도중에는 브라우저를 이용하여 Tensorflow Documentation만을 참고 가능합니다.
  2. 반드시 혼자 시험을 치루셔야 합니다.
  3. 시험을 치르는 데에는 100불이 소요됩니다.

재시험 관련

  1. 1번째 시험 낙방 -> 2주가 지나야 시험을 칠 수 있습니다.
  2. 2번째 시험 낙방 -> 2달이 지나야 시험을 칠 수 있습니다.
  3. 3번째 시험 낙방 이후 -> 1년이 지나야 시험을 칠 수 있습니다.
